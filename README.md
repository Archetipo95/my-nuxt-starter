# Nuxt Starter

Description

- Site: url
- Storybook: url

## Features

1) Nuxt 3
2) TailwindCSS
3) Nuxt VueUse
4) Storybook 6
5) Nuxt lodash

## To be added

1) Icons support

## Setup

Make sure to install the dependencies:

```bash
# yarn
yarn install
```

## Development Server

Start the development server on http://localhost:3000

```bash
yarn dev
```

## Production

Build the application for production:

```bash
yarn build
```
